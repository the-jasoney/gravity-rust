# gravity-rust
## build instructions
### prerequisites
- rust
- cargo

### dev
run
```sh
cargo run
```

## controls
Left-click and Drag: create new ball with velocity towards drag direction

<kbd>Esc</kbd> - Exit

<kbd>Backspace</kbd> - Delete all objects

<kbd>Space</kbd> - Toggle show vectors

<kbd>P</kbd> - Toggle show projection
